## About

ZeroPoint is a project that is improving the old private server [Aurora](https://AuroraFN.dev) made by [Cyuubi](https://twitter.com/nyameows) and [Slushia](https://twitter.com/br1ia) and updated by [Jacobb](https://twitter.com/Jacobb626_), [Beat YT](https://twitter.com/TheBeatYT_evil), [Rythm](https://twitter.com/Kian62067823) and [.PAK](https://twitter.com/FTLeaks_)

## About the server

The backend is made in [Node.js](https://nodejs.org/) a very powerful runtime of JavaScript.
The backend uses a node module called Express.js in order to run the api server.
The backend xmpp server is built in a websocket server.
